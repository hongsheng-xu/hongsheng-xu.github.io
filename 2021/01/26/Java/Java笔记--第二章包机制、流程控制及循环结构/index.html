包机制

包的本质是文件夹,用于区别类名的命名空间

一般使用公司域名倒置做为包名

//www.baidu.com其包名为com.baidu.www
//项目文件右上角设置里取消勾选Compact Middle Package
package com.baidu.www; //在最上面
import java.util.Date; //导入包,语句必须在package下面
import com.baidu.*;//导入所有类

java的Scanner对象–与用户交互

Scanner基本语法:

Scanner s= new Scanner(System.in);
//通过Scanner类的next()与nextLine()方法获取输入的字符串,在读取输入数据以前需要使用hasNext()或hasNextLine()判断是否还有输入的数据
package xyz.honshen;
import java.util.Scanner;
public class HelloWord {
    public static void main(String[] args) {
        Scanner s = new Scanner(System.in);
        System.out.println("next接收输入: ");
        //判断用户是否输入
        if (s.hasNext()) {
            String str = s.next();  //next方式接收,只能到空格结束
            //println会换行,print不会换行
            System.out.println("输入内容为: " + str);    //输出输入内容
        }
        System.out.println("nextline接收输入: ");
        if (s.hasNextLine()){
            String str = s.nextLine(); //nextLine方式接收,接收一行到\n结束
            System.out.println("输入内容为: "+ str); //输出输入内容
        }
        s.close(); //关闭Scanner io流必须关闭否则会占用资源
    }
}

java程序之顺序–选择–循环结构(break,continue)

顺序结构

  • 一句一句执行,从上往下

    //从上到下输出1 2 3
public static void main(String[] args) {
        System.out.println("1");
        System.out.println("2");
        System.out.println("3");
    }

选择结构

  • if选择结构

    String s = "abc"
//string.equals("str")判断字符串是否一致
//if语句的嵌套
if(s.equals("abc")){
    //执行语句
    if(a<b){
        //执行语句
    }
}
//多个判断
if(a<b){
    //执行语句1
}else if(a<c){
    //执行语句2
}else if(a<d){
    //执行语句3
}else{
    //执行语句4
}

  • switch选择结构

    //从javase7开始支持比较字符串,本质是比较数字(hash数字)
switch(表达式){
    //本质是hashcode编码
    /*
    在文件路径下输入cmd+空格然后回车可直接进入cmd
    将编译的class文件用idea直接打开获得反编译文件反编译后结果,如下
    String name = "abc";
    switch(name.hashCode()){
        case 9987: 
            //语句
            break;
    }
    */
    case "中国":
        //语句
        break;
    //不加break,则case穿透会一直执行到break
    case value: 
        //语句
        break;
    default value://可选
        //语句
        break;
}

循环结构

  • while

    while(布尔表达式){
    //执行语句
}
while(true){
    //无限循环,常用于服务器监听
}

  • do-while

    do{
    //执行语句,至少执行一次
}while(布尔表达式)

  • for

    for(初始化;布尔表达式;更新){
    //执行语句
}
//例如:
for(int i=0;i<100;i++){
    //执行语句
    //在idea输入100.for可以直接获得100次的for循环结构
}

  • 增强型for循环

    //jdk5引入的特性,主要遍历数组和集合
for(声明语句:表达式){
    //执行语句
}
//例如数组遍历: 
int[] numbers = {10,20,30}; //定义一个数组
for (int i=0; i<3; i++){
    numbers[i]; //每次循环numbers[i]为: 10,20,30
}
for (int val : numbers){
    val;//每次循环val为: 10, 20, 30
}

break和continue 组合成类似goto语句

  • break用于循环和switch,强制退出循环或语句块
  • continue只用于循环,跳过本次循环,进入下一次循环
  • goto语句:java中保留字含有goto,但并未得到使用,给语句加上label,通过break和continue实现(不建议使用了解即可)
while(true){
    if(布尔表达式)
        break;//结束循环
    if(布尔表达式)
        continue;//进入下一次循环
    //执行语句
}
//利用label求质数
outer: for (int i=1;i<100;i++){
            for(int j=2;j<i/2;j++){
                if (i%j==0) {//有其他因数,则不是质数
                    continue outer;
                }
            }
            System.out.println(i);
        }
