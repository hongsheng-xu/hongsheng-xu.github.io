# break 和 continue 组合成类似 goto 语句

  • break 用于循环和 switch, 强制退出循环或语句块
  • continue 只用于循环，跳过本次循环，进入下一次循环
  • goto 语句:java 中保留字含有 goto, 但并未得到使用，给语句加上 label, 通过 break 和 continue 实现 (不建议使用了解即可)
