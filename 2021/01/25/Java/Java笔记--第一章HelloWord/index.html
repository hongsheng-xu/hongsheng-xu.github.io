Tips:

  1. 文件名与类名一致 (类名大写，这是规范), 且在该文件的目录下执行 java 命令

  2. 大小写敏感

  3. javac 文件名.java -- javac 生成的是文件名.class 文件

  4. java 文件名 -- java 执行的是文件名.class 文件，执行 java 命令文件名不用带后缀 class
