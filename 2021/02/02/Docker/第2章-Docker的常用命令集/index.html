# 容器命令

  • # 运行镜像

    参数及作用

    可选参数作用
    --name="容器名称"为容器设置名称用于区分容器
    -d以后台方式运行，但是会暂停运行
    -it使用交互方式运行: docker run -it centos /bin/bash
    -p指定端口，例如 -p 3344:80
    -P随机指定端口

    docker 容器后台运行需要一个前台进程，docker 会停止后台应用

  • # 查看运行的镜像

    参数及作用

    参数作用
    -a列出当前运行的容器以及历史运行过的容器
    -n = 数字显示最近创建的几个容器常用: -n=1
    -q只显示容器 id 常用: docker ps -aq 列出所有容器 id
